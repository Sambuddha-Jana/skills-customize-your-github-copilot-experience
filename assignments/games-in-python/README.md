
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game in Python that uses strings, loops, conditionals, and user input. Students will practice program flow control, state tracking, and basic I/O.

## 📝 Tasks

### 🛠️ Game Core

#### Description
Implement the main Hangman gameplay loop. The program should choose a secret word, accept single-letter guesses from the player, reveal correct letters in place, and track remaining attempts for incorrect guesses.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list (included in `starter-code.py`).
- Accept single-letter input and validate it (ignore repeated guesses and non-letter input).
- Display the current word progress (for example: `_ a _ _ n`).
- Track and display the number of incorrect guesses remaining.
- End the game with a clear win or lose message and show the secret word when the player loses.


### 🛠️ Optional Enhancements

#### Description
Add one or more optional features to improve the game experience.

#### Requirements (optional)

- Add difficulty levels that change the number of allowed incorrect guesses.
- Load the word list from an external file (for larger vocabularies).
- Display simple ASCII-art hangman stages for incorrect guesses.
- Keep score across multiple rounds or implement a hint system.


## Starter Code

A starter file `starter-code.py` is provided to help you get started. Use it as scaffolding rather than a final solution.

## Submission

Submit your final script (`starter-code.py` or a new file) and any additional resources (word lists) in the `assignments/games-in-python/` folder.
